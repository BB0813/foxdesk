FoxDesk

本地 Camoufox 指纹浏览器管理器 — 免费、开源、隐私优先

Python 3.12+ FastAPI Camoufox Windows / macOS / Linux

安装与启动

方式一：直接运行（推荐开发者）

git clone https://github.com/yourname/foxdesk.git
cd foxdesk
python -m pip install -r requirements.txt
python desktop.py

或双击 Start-CamoufoxManager.bat

方式二：下载 exe 安装包

从 GitHub Releases 下载最新版本，解压后双击 FoxDesk.exe 即可运行。

无需安装 Python 环境。

方式三：Web 调试模式

python -m uvicorn backend.app:app --host 127.0.0.1 --port 8765 --reload
# 浏览器打开 http://127.0.0.1:8765

首次启动

首次启动需要安装 Camoufox 浏览器。进入「系统」标签页，按顺序执行：

1. Install — 安装 Python 包 pip install camoufox

2. Fetch — 下载浏览器二进制文件 camoufox fetch

3. Test — 运行自检（可选，可能耗时较长）

快速上手

  1. 启动 FoxDesk
  2. 点击「新建」创建一个 Profile
  3. 在「基础」标签页设置名称和启动网址
  4. 在「代理」标签页填写代理地址（可选）
  5. 在「指纹环境」标签页点击「随机生成」
  6. 点击「保存」
  7. 点击「启动」打开浏览器

配置档案

Profile 是 FoxDesk 的核心概念，每个 Profile 对应一个独立的浏览器身份。

字段说明默认值
name档案名称新档案
startup_url浏览器启动后打开的网址browserleaks.com
mode浏览器模式 / 远程服务模式browser
os模拟的操作系统auto
user_data_dir浏览器数据持久化目录自动生成
locale语言区域空（使用系统默认）
tags自定义标签，逗号分隔空

操作

  • 新建 — 创建空 Profile
  • 保存 — 保存当前编辑的 Profile
  • 复制 — 克隆当前 Profile
  • 删除 — 右键菜单删除
  • 导入/导出 — JSON 格式导入导出
  • 右键菜单 — 在 Profile 列表右键可启动、编辑、复制、导出、删除

随机生成

点击「随机生成」按钮，会根据选择的 OS 自动生成一套完整的指纹参数组合。

指纹检测

点击编辑器顶栏「检测」按钮，FoxDesk 会分析当前 Profile 的指纹一致性，给出 0-100 分评分并列出潜在问题。

代理设置

支持的代理协议：

协议格式
HTTPhttp://host:port
HTTPShttps://host:port
SOCKS4socks4://host:port
SOCKS5socks5://host:port

支持认证：socks5://user:pass@host:port

只填 host:port 会自动补全为 http://host:port

代理测试

点击「测试代理」按钮，FoxDesk 会通过代理连接到检测服务器，返回：

  • 出口 IP 地址
  • 连接延迟
  • 成功/失败状态

批量导入

在代理标签页底部，每行一个代理地址，点击「批量导入」会自动分配给 Profile。

会话管理

每个启动的浏览器是一个「会话」。在「运行会话」标签页可以看到所有运行中的会话。

  • 展开详情 — 点击会话行的下拉箭头
  • 查看日志 — 展开后可看到实时日志
  • 下载日志 — 点击「下载日志」按钮
  • 停止 — 点击「停止」按钮终止进程
  • 全部停止 — 一键停止所有会话

Cookie 管理

在「指纹环境」标签页底部：

  • 导出 Cookie — 从 Profile 的 cookies.sqlite 读取并下载为 JSON
  • 导入 Cookie — 上传 JSON 格式的 Cookie 文件

数据流

桌面启动流程：

  1. desktop.py 找可用端口
  2. 启动 uvicorn 后端（子进程或内嵌）
  3. 等待后端就绪
  4. 用 pywebview 打开窗口（或回退到浏览器）

会话启动流程：

  1. 前端调用 POST /api/sessions
  2. 后端验证 Profile → 写入 runtime JSON
  3. 启动 camoufox_worker.py 子进程
  4. Worker 读取 Profile → 调用 Camoufox API
  5. Worker 通过 stdout 输出 JSON 日志事件
  6. 后端捕获日志 → 前端轮询显示
